## AWS Roles
### Create new role
<p>Path: AWS Console > IAM > Roles > Create role</p>
<p>Add all three roles:</p>
<p>AWSElasticBeanstalkWebTier</p>
<p>AWSElasticBeanstalkWorkerTier</p>
<p>AWSElasticBeanstalkMulticontainerDocker</p>
